## Developing

```
$ git clone https://github.com/sircus/docs.git sircus-docs
$ cd sircus-docs
$ npm run setup
```

build:

```
$ gulp build
```
```
$ npm run build
```

Watch & Connect:

```
$ gulp
```

deploy:

```
$ gulp deploy
```
